# 🌾 Crop Disease Detection Using AI

## 📌 Project Overview

This project focuses on detecting crop diseases using Artificial Intelligence (AI) and Machine Learning techniques. The system analyzes images of crops (such as rice leaves) and predicts whether the crop is healthy or affected by a disease. This helps farmers take early action and improve crop yield.

---

## 🎯 Objectives

* Detect crop diseases from images using AI models
* Classify images into healthy or diseased categories
* Provide a simple and efficient solution for farmers
* Reduce crop loss through early detection

---

## 🧠 Technologies Used

* Python
* Jupyter Notebook
* TensorFlow / Keras
* NumPy
* Pandas
* Matplotlib
* OpenCV

---

## ⚙️ How It Works

1. **Data Collection** – Images of crops (healthy and diseased) are collected
2. **Preprocessing** – Images are resized and normalized
3. **Model Training** – A deep learning model (CNN) is trained on the dataset
4. **Prediction** – The trained model predicts disease from new images
5. **Evaluation** – Accuracy and performance are measured

---
